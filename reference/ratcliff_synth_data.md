# A synthetic data set with one condition

This dataset was simulated by using the classical Ratcliff diffusion
model (see
[`ratcliff_dm()`](https://bucky2177.github.io/dRiftDM/reference/ratcliff_dm.md)).

## Usage

``` r
ratcliff_synth_data
```

## Format

A data frame with 300 rows and 3 columns:

- RT:

  Response Times

- Error:

  Error Coding (Error Response = 1; Correct Response = 0)

- Cond:

  Condition ('null')
