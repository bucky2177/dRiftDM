# Count the number of digits \> 0

This function takes one entry of the internal_list (i.e., all conditions
for one parameter) and counts how often there are unique digits \> 0

## Usage

``` r
count_unique_prms_one_internal_entry(one_internal_entry)
```

## Arguments

- one_internal_entry:

  such as `internal_list[["A"]]`

## Value

an integer number
